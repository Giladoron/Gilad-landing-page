## Image Upload Instructions

1. **Image Format:** WebP (recommended), JPG, or PNG
2. **Image Location:** Upload to `/public/assets/results/` folder
3. **Naming:** Use the filenames you specified above (e.g., `result-01.webp`, `result-02.webp`, etc.)
4. **Image Content:** Before/after comparison images (side-by-side or split view)
5. **Image Size:** Optimize for web (recommended max width: 1000px)

---

## Notes

- **Quote:** Keep it to 2-3 sentences max (will be truncated to 2 lines in the carousel)
- **Goals:** List 2-3 main goals
- **Stats:** Use format like "+3 ק\"ג" for gains, "-12 ק\"ג" for weight loss, "+40%" for percentage increases
- **Image filenames:** Make sure they match exactly what you write in the template
